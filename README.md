# AnchorDesk (AnchorDesk)

构建一个面向个人用户和小团队的工作空间级 AI 助手。系统以用户私有资料库为主，必要时联网补充公开信息；所有关键回答默认要求可追溯、可点击、可核验。

## 产品定位

- **基于证据回答**：先检索再生成，给出依据后给出结论。
- **工作空间级隔离**：资源和上下文按工作空间完全隔离。
- **可支持人工接管**：提供工作空间内完整的资料与对话管理。

## 核心场景

1. **工作空间问答**：基于私有资料库的提问与解答，支持引用溯源。
2. **外部公开信息补充**：网络搜索补充，明确区分本地依据与外部来源。
3. **长文结果生成**：大纲生成、内容整理、报告导出。
4. **资料上传与消化**：支持多格式（PDF、DOCX、TXT、Markdown）的解析、切块与检索，内置完整任务状态追踪。
